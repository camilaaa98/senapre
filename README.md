# AsistNet - Sistema de Gestión de Asistencias SENA

Sistema nativo en HTML, CSS y JavaScript para la gestión de asistencias del Servicio Nacional de Aprendizaje (SENA) de Colombia.

## Características

- Login con validación en tiempo real
- Panel de Administrador con 7 módulos
- Panel de Instructor con 4 módulos
- Diseño responsive y moderno
- Animaciones suaves y efectos visuales
- Colores institucionales del SENA
- Sistema de autenticación con localStorage
- Partículas de fondo animadas

1. Descarga todos los archivos
2. Mantén la estructura de carpetas
3. Abre `index.html` en tu navegador
4. ¡Listo para usar!
